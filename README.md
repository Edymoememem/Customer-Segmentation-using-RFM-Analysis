# 📊 Customer Analytics Dashboard — Syntecxhub

A multi-page Power BI dashboard built to analyze customer demographics, segmentation, and sales performance across multiple countries and regions.

---

## 📌 Project Overview

This project explores customer data to uncover patterns in purchasing behavior, demographic profiles, and customer segments. The dashboard was built as part of my data analytics internship at **Syntecxhub** and demonstrates end-to-end reporting using Power BI.

---

## 📂 Pages & Features

| Page | Description |
|------|-------------|
| **Customer Demographic** | Breakdown of customers by age, marital status, gender, and location |
| **Customer Segmentation** | RFM-based segmentation — Champions, Promising, Potential Loyalists, Cannot Lose Them |
| **Customer Details** | Full customer records table with drill-through capability |
| **Page 4** | Additional analysis and visual summaries |

---

## 📊 Key Metrics & Visuals

- **Total Sales** by country and city (Map visual)
- **Customer Segments** — Champions, Promising, Potential Loyalists, Cannot Lose Them
- **Age distribution** across customer base
- **Marital status** breakdown
- **Top customers** by total sales
- **Geographic spread** across Australia, United States, and more

## 💡 Key Insights

- Majority of high-value customers are located in **Australia**
- **Champions** segment drives the highest total sales
- Customers aged **55–65** represent the largest demographic group
- **Married customers** account for a significant portion of top spenders

1. Download and install [Power BI Desktop](https://powerbi.microsoft.com/desktop) (free)
2. Clone or download this repository
3. Open `Syntechx2.pbix` in Power BI Desktop
4. Explore the dashboard pages using the page tabs at the bottom
